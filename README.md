# Poker Clone

## Description
-  A poker clone where the user can play texas hold em

## APIs
- Deck of Cards [api](https://www.deckofcardsapi.com/)
- [PokerSolver](https://github.com/goldfire/pokersolver)
  

## Tech Stack

- React
- Node
- Express
- MongoDB
- WebSocket.io
- Bootstrap
- Mongoose
- Axios
- React Router DOM
